Group Object
============

The Group object is a logical container for a number of Blocks.  It has an id
which is for internal use only, a title, some display properties and an array
of Blocks

.. code-block:: javascript
    :linenos:

    {
        "blocks": [...],
        "display": {
            "properties": {}
        },
        "id": "14ba4707-321d-441d-8d21-b8367366e766",
        "title": ""
    }
